# tetris
Enjoy a game of tetris!

## Tech Stack

**Client:**
* [React](https://reactjs.org/)
* [Redux](https://redux.js.org)

**Server:** 
* [Webpack](https://webpack.js.org/)
* [Node](https://nodejs.org/en/)
* [Express](https://expressjs.com/)

## Running tetris

1. Install dependencies with 
             
        npm install
                

2. Run the application

        npm run dev
